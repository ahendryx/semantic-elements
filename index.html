Semantic Elements

WHAT ARE THEY?

In HTML, semantic elements (or tags) are used to define the different components of a webpage. The intention of these tags is to clearly identify the content they include rather than to apply any sort of style to it. It is preferred that style is applied to elements using CSS.

WHAT ELSE ARE THEY GOOD FOR?

  • Using semantic tags can make the reasoning behind your page structure more evident to any other developers who may need to work with your code.
  • Another advantage is that they help assistive technologies interpret the content accurately, making your site more accessible.
